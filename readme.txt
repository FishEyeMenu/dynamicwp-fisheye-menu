== Description ==

DynamicWP fisheye menu is a jQuery wordpress plugin, this plugin will build a Fisheye menu from a list of links. Like Mac OS user interface at the bottom of the screen, where there is a dockbar with icons on it, and when you mouse over the icons they grow in size, giving the “fisheye” effect.
This plugin using a script to displayed, so you can put it anywhere you like.

**Features**

* Easy to use
* Navigation menus with fisheye effect
* Can appear anywhere

**Demo**

Please check [Fisheye Menu demo page](http://www.dynamicwp.net/fisheye-menu-demo/).

== Installation ==

1. Decompress the .zip archive and put the “dynamicwp-fisheye-menu” folder into your plugins directory (/wp-content/plugins/).
1. Or the other way, with your wordpress admin page > Plugins > Add New then choose upload on the top. Next choose File (.zip) then Install Now. (see the screenshots)
1. Activate the plugin in the WordPress Plugins admin page.
1. Now you can see Fisheye menu below the Settings menu. (see the screenshots)
1. How to usage this plugin, please go to the plugin usage section [here](http://www.dynamicwp.net/plugins/dynamicwp-fisheye-menu/)
